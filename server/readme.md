# Benchmark Server

Flask API server for the benchmark application.

## Quick Start

1. **Install dependencies:**

   ```bash
   uv sync
   ```

2. **Run the server:**
   ```bash
   uv run python app.py
   ```

The server will start at `http://127.0.0.1:5000`

## Development

Run in debug mode:

```bash
uv run flask run --debug
```
